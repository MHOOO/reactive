Getting Started with reactive-web

  1. Install sbt

  2. Check out the sources and publish locally

      git clone git://github.com/nafg/reactive.git
  cd reactive
  sbt publish-local

  3. Choose an existing Lift sbt project or create a new one

  4. Add reactive-web to the dependencies

    In project/build/Project.scala (or whatever your sbt project file is called), for example:

      override def libraryDependencies = Set(
    "reactive" %% "reactive-web" % "0.1",
    //...
  ) ++ super.libraryDependencies

  5. Optionally set up Comet support

    In Boot.scala:

      Reactions.initComet

    Once on each page, before a snippet that uses Comet support (may be in individual templates or in your surround template, it doesn't matter):

      <div class="lift:reactive"></div>

  6. Start coding!

    • You will probably want to import reactive._ and reactive.web._
    • Snippets may need to extend Observing
    • See the rest of this site for more information
