Forwardable

Several types extend Forwardable . This gives you a convenient syntax, >> , to act on such objects' values in several common ways:

  • By running a parameterless block of code
  • By passing the value to a function
  • By assigning the value to a Var
  • By firing the value from an EventSource.

Types that extend Forwardable include EventStream , Signal , and reactive-web's DOMEventSource . 

click >> println("Mouse was clicked!")
intEventStream >> {i: Int => println("Caught an event: "+i)}
aVar >> anotherVar
myVar >> myEventSource

It should be noted that forwarding can lead to more imperative code when often a functional solution is available and more desirable.
